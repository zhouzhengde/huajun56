公司简介 

        深圳市前海华军物流有限公司是经深圳市工商局批准，注册资金人民币888.88万元；以深圳为中心，公司主营国内公路干线运输、铁路运输、空运代理、海运运输、仓储配送、搬家包装、市内配送、设备迁移、到达派送、提供物流信息咨询及物流方面设计为一体的综合性物流企业。同时，华军物流是全国领先的物流服务咨询商，业务范围覆盖于：北京、天津、济南、石家庄、西安、太原、武汉、郑州、呼和浩特、兰州、银川、西宁、成都、重庆、昆明、贵阳、拉萨、乌鲁木齐、南宁、合肥、南京、苏州、无锡、常州、福州、厦门、广州、深圳、海口、三亚等市级县级地区，并在全国各地加盟了近千家第三方专线联盟合作单位。公司自备及合同签约集装箱货车、配送车千余部和各种装卸机械、停车场，并且全公司实行电脑网络自动化动作。伴随着市场的发展和建立，华军掀开了企业发展进程中新的篇章。

        华军物流以“专业服务，专心专注”为理念，重视知识创新与变革，将高新信息科技与传统物流行业的优良经验相结合，秉承物流现代标准化概念，跃身现代物流行业标杆企业之一。华军物流以共赢为终极目标，已得到了众多一线品牌客户的认可和支持，奠定了华军稳健、迅速发展的基石。
